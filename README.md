# Google Voice 转移到其他 Google 账户下

1. 打开 [Google Voice](https://voice.google.com/u/0/messages)，点击右上角设置按钮。
   
   

![](pic/01.png)

2. 点击「转移」。


![](pic/02.png)


3. 点击「Transfer」。



![](pic/03.png)



4. 点击「Sign-in to another account」。



![](pic/04.png)



5. 登陆新 Google 账户。



![](pic/05.png)



6. 点击「Transfer」。



![](pic/06.png)



7. 选择新 Google 邮箱，点击「Transfer」，完成操作。

> 两个 Google 账户需要同时在线。



![](pic/07.png)

提示下图表示 Google Voice 已经成功转入新 Google 账户内。

![](pic/08.png)
